# Contributing to Game Capture

Thanks for helping improve Game Capture.

## Pull Requests

- Fork the repo and create a feature branch.
- Keep changes focused and include tests where practical.
- Run local build/test checks before opening a PR.
- Open a PR with a clear summary of what changed and why.
